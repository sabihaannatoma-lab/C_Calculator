 # Scientific calculotor in C 
 This is a beginner-freindly scientific calulator project written in C
 ## Features
 Basic arithmetic operations (Adiition,Subtraction,Multiplication,Division)
 Scientific funtions:
  *Square Root
   *power(x^y)
   *Natural Log (ln)
   *Log Base 10
   *Trigonometric functions(sin, cos, tan)
Menu-drivven and loop based
Input validation for safe calculations
# How to run
complile the program using a C compiler.
run the program.
## Purpose
This project was created to practice C programming concepts such as:
*Loop
*Switch-case
*Functions from math.h
*Error handling
